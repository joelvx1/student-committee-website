Accessibility Statement

This is an accessibility statement from University of Sheffield.

University of Sheffield takes the following measures to ensure accessibility of Staff Student Liaison Committee: include accessibility as part of our mission statement, integrate accessibility into our procurement practices. Conformance status: The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. Staff Student Liaison Committee is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard. Feedback: We welcome your feedback on the accessibility of Staff Student Liaison Committee. Please let us know if you encounter accessibility barriers on Staff Student Liaison Committee: E-mail: com-teaching@sheffield.ac.uk. We try to respond to feedback within 2 business days. Technical specifications Accessibility of Staff Student Liaison Committee relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer: HTML, CSS. These technologies are relied upon for conformance with the accessibility standards used.

Self-evaluation Date: This statement was created on 14 November 2023 using the W3C Accessibility Statement Generator Tool.

Accessibility Choices

I have used HTML5 semantic elements to make it easier for screenreaders to parse this website and therefore make it more accessible. This also makes it easier for the website to be navigated using just a keyboard (tab key). All images have been provided with appropriate alt-text. Every element on this website has been checked to see if it conforms with WCAG 2 's contrast guidelines.
